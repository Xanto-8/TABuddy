# TABuddy 班级管理员使用说明

欢迎您成为 **TABuddy 班级管理员**！作为班级管理员（任课老师），您拥有比普通助教更高的管理权限，可以管理助教团队、维护公共知识库、生成老师注册码等。本手册将详细介绍您作为班级管理员的专属功能。

## 1. 角色概述

### 1.1 班级管理员是什么

班级管理员（classadmin）是 TABuddy 系统中的**任课老师角色**，位于超级管理员和助教之间：

| 角色 | 定位 | 核心职责 |
|------|------|---------|
| 超级管理员（superadmin） | 平台运维者 | 用户管理、班级组织管理、全局配置 |
| **班级管理员（classadmin）** | **任课老师** | **教学管理、助教团队管理、知识库维护** |
| 助教（assistant） | 日常教学执行者 | 作业批改、测验记录、反馈生成 |

### 1.2 班级管理员独有权限

作为班级管理员，您可以使用以下**专属功能**：

- **助教与绑定管理** — 生成邀请码，邀请助教加入您的班级团队
- **已绑定成员管理** — 查看和管理您绑定的助教账号信息
- **公共知识库管理** — 维护全局公共知识库，所有用户共享
- **老师注册码管理** — 生成注册码邀请其他老师成为班级管理员

### 1.3 权限范围说明

| 功能 | 班级管理员 | 说明 |
|------|-----------|------|
| 公共知识库 | 可管理 | 增删改查公共知识条目 |
| 班级知识 | 可管理 | 管理自己班级范围内的知识 |
| 班级数据统计 | 可查看 | 查看自己班级的教学统计数据 |
| 助教邀请码 | 可生成 | 生成邀请码绑定助教 |
| 绑定成员管理 | 可管理 | 修改已绑定助教的账号信息 |
| 老师注册码 | 可生成 | 生成注册码邀请其他老师 |
| 用户管理 | 无权限 | 由超级管理员负责 |
| 班级组织管理 | 无权限 | 由超级管理员负责 |
| IP封禁管理 | 无权限 | 由超级管理员负责 |
| 全站统计 | 无权限 | 由超级管理员负责 |

## 2. 快速入门

### 2.1 成为班级管理员

有以下两种方式成为班级管理员：

**方式一：通过老师注册码注册**
1. 在登录页点击「注册」标签
2. 输入由超级管理员或其他班级管理员提供的**老师注册码**
3. 填写用户名、密码，完成注册
4. 系统自动赋予您班级管理员身份

**方式二：由超级管理员设置**
- 联系超级管理员，在「用户管理」中将您的账号角色设置为「班级管理员」

### 2.2 登录系统

在登录页输入您的**用户名**和**密码**，点击「登录」即可进入系统。

### 2.3 界面概览

登录后您将看到：

- **左侧边栏**：包含通用功能导航和班级管理员专属导航（有分隔线区分）
- **右侧内容区**：显示当前功能页面
- **右下角气泡**：AI 浮动助手「小T」

侧边栏中，班级管理员专属的导航区域包含以下入口：

| 菜单项 | 说明 |
|-------|------|
| 助教与绑定管理 | 生成邀请码、管理已绑定助教 |
| 已绑定成员管理 | 查看和管理助教账号详细信息 |
| 公共知识库管理 | 维护公共知识条目 |
| 老师注册码 | 生成注册码邀请其他老师 |

## 3. 通用教学功能

作为班级管理员，您同样可以使用所有核心教学功能。以下是简要说明：

### 3.1 仪表盘（首页）

日常工作起点，汇集所有核心数据：
- **欢迎条幅**：显示您的姓名、角色和今日核心统计
- **班级待办中心**：按班级列出作业待批改、测验待记录、任务待完成数量
- **课程表与考勤**：今日/本周/本月视图，支持标记调课/停课
- **班级学习概览**：折线图展示各班级正确率变化趋势
- **工作统计**：今日各类型任务完成情况
- **重点关注学生**：展示被标记为「重点」的学生名单
- **专注计时器**：内置番茄钟，辅助时间管理

### 3.2 班级管理

- **查看班级列表**：以卡片形式展示您管理的所有班级
- **班级详情**：管理上课时间、课程任务模板、课时进度
- **核心数据概览**：学生人数、今日记录数、工作流完成统计

### 3.3 学生管理

- 查看/添加/编辑/删除学生
- 按班级筛选学生
- 标记「重点关注」学生
- 记录上课缺勤

### 3.4 作业管理

- 记录作业评估（完成度、正确率、书写质量）
- 支持 AI 辅助生成评语
- 支持导出 Excel/Word

### 3.5 随堂测验

- 记录测验成绩（单词分、正确率）
- 管理重测名单
- 支持导出 Excel/Word

### 3.6 课后反馈

- 基于预置模板自动生成个性化反馈
- 支持导出 Word 文档

### 3.7 工作流系统

- 为每种课程类型定义标准化教学流程
- 系统自动为每个上课日生成待办事项
- 支持自定义节点和模板

### 3.8 其他通用功能

- **知识库管理**：管理您个人的私有知识条目
- **教学资源**：管理课件、文件、链接等
- **AI 浮动助手**：智能问答、快捷导航、智能体操作
- **设置**：个人信息修改、主题切换、数据备份与恢复
- **多账号管理**：支持多个账号共存，数据完全隔离

> 以上通用功能的详细操作说明，请参考《TABuddy 英语助教系统 — 使用说明书》。

## 4. 助教与绑定管理

这是班级管理员的核心功能，用于生成邀请码并邀请助教加入您的教学团队。

### 4.1 进入页面

在侧边栏「班级管理员」分区中，点击 **「助教与绑定管理」**。

### 4.2 生成邀请码

1. 在页面中点击 **「生成邀请码」** 按钮
2. 选择邀请码的有效期：
   - **永久有效** — 邀请码长期有效，可重复使用
   - **24小时有效** — 邀请码在24小时后失效
   - **7天有效** — 邀请码在7天后失效
3. 系统自动生成一个唯一的邀请码
4. 点击 **「复制」** 按钮将邀请码发送给您的助教

### 4.3 邀请码说明

- 每个班级管理员同时只能拥有**一个有效的邀请码**
- 如已有一个邀请码，再次生成会**替换**旧的邀请码（旧码立即失效）
- 邀请码可以**重复使用**，多个助教可以使用同一个邀请码绑定
- 助教使用邀请码注册或绑定后，自动获得您名下**所有班级**的查看权限
- 后续您新创建的班级，已绑定的助教也会**自动同步**获得可见权限

### 4.4 查看已绑定助教

在「助教与绑定管理」页面下方，您可以：

- 查看所有已绑定助教的列表（用户名、显示名称、绑定时间）
- 点击 **「移除」** 按钮解绑某个助教
  - 解绑后，该助教将无法再查看您的班级数据
  - 解绑操作不可逆，请谨慎操作

### 4.5 助教绑定流程（供您指导助教参考）

1. 助教在登录页点击「注册」
2. 在注册表单中输入您提供的**邀请码**
3. 填写用户名和密码完成注册
4. 注册成功后，助教即可在系统中查看您名下的所有班级数据
5. 如助教已有账号，可在「设置」→「账号管理」中输入邀请码进行绑定

## 5. 已绑定成员管理

此功能用于查看和管理已绑定助教的账号详细信息。

### 5.1 进入页面

在侧边栏「班级管理员」分区中，点击 **「已绑定成员管理」**。

### 5.2 查看成员列表

- 页面以列表形式展示所有已绑定助教的账号
- 支持**搜索** — 按用户名或显示名称快速查找

### 5.3 查看成员详情

点击某个成员行，可以展开查看详细信息：

- **账号信息**：用户名、显示名称、密码明文
- **注册时间**：该账号的注册日期
- **最后活跃时间**：最近一次登录时间
- **在线状态**：当前是否在线
- **登录IP**：最近登录的IP地址及版本类型
- **登录地理位置**：最近登录的地理位置

> 在线状态每8秒自动刷新一次。

### 5.4 编辑成员信息

在成员详情中，您可以：

- **修改显示名称**：为助教设置更易识别的昵称
- **重置密码**：为助教设置新的登录密码

这些操作对助教来说是**即时生效**的。

## 6. 公共知识库管理

公共知识库是系统所有用户共享的知识资源，作为班级管理员，您可以维护这些公共知识条目。

### 6.1 进入页面

在侧边栏「班级管理员」分区中，点击 **「公共知识库管理」**，或直接通过侧边栏的「管理」→「公共知识库管理」进入。

### 6.2 查看知识条目

页面以列表形式展示所有公共知识条目，每条包含：

- **标题**：知识的名称
- **类型**：链接(link)、模板(template)、文档(document)、信息(info)
- **关键词**：用于智能匹配的搜索关键词
- **内容**：知识的详细内容
- **链接URL**：关联的外部网址（仅链接类型）
- **优先级**：1-10 的数字，越高越优先匹配
- **启用状态**：该条目是否对用户可见

### 6.3 搜索条目

支持按**标题、内容、关键词**进行搜索，快速定位所需知识条目。

### 6.4 添加条目

1. 点击 **「添加条目」** 按钮
2. 填写以下信息：
   - 标题（必填）
   - 类型（必选）
   - 关键词（用逗号分隔）
   - 内容（必填）
   - 链接URL（可选）
   - 优先级（1-10，默认为5）
3. 点击确认保存

### 6.5 编辑条目

点击条目旁的编辑按钮，修改已有知识条目的任何字段。

### 6.6 启用/禁用条目

- 点击条目旁的开关按钮，可以控制该条目是否对用户可见
- 禁用的条目不会在知识匹配和AI助手的回答中出现

### 6.7 删除条目

点击条目旁的删除按钮，确认后即可移除该条目。

### 6.8 恢复默认知识库

点击 **「恢复默认」** 按钮，将所有公共知识条目重置为系统预置的默认内容。

> 注意：此操作会覆盖所有您已添加或修改的公共知识条目，请谨慎使用。

### 6.9 诊断面板

页面底部提供诊断面板，可以查看：
- 公共知识的Token状态
- 与服务器的同步状态
- 服务器Redis的连接状态

## 7. 老师注册码管理

通过老师注册码，您可以邀请其他老师注册成为班级管理员，壮大您的教学团队。

### 7.1 进入页面

在侧边栏「班级管理员」分区中，点击 **「老师注册码」**。

### 7.2 生成注册码

1. 点击 **「生成注册码」** 按钮
2. 系统自动生成一个唯一的老师注册码
3. 点击 **「复制」** 将注册码发送给其他老师

### 7.3 管理注册码

- **查看注册码列表**：显示所有已生成的注册码及其状态
- **启用/禁用**：控制注册码是否可用
  - 禁用后，使用该注册码注册将无法成功
  - 可随时重新启用
- **重置**：生成新的注册码替换当前激活的注册码

### 7.4 注册码说明

- 通过老师注册码注册的账号，**自动成为班级管理员（classadmin）身份**
- 注册码可**重复使用**，多个老师可以使用同一个注册码注册
- 注册码可以随时启用或禁用，灵活控制注册入口

## 8. 常见问题

### Q：邀请码和老师注册码有什么区别？

A：两者的用途不同：

| 类型 | 用途 | 注册后的角色 |
|-----|------|-------------|
| 邀请码 | 邀请助教加入您的班级团队 | 助教（assistant） |
| 老师注册码 | 邀请其他老师注册系统 | 班级管理员（classadmin） |

### Q：助教绑定后能看到什么数据？

A：助教绑定后，可以看到您名下**所有班级**的数据，包括班级详情、学生信息、作业记录、测验成绩、课后反馈等。您后续新增的班级也会自动对已绑定的助教可见。

### Q：如何解绑某个助教？

A：进入「助教与绑定管理」，在已绑定助教列表中找到该助教，点击「移除」按钮即可解绑。解绑后，该助教将无法再查看您的班级数据。

### Q：公共知识库和私有知识库有什么区别？

A：私有知识库只有您自己能看见和使用；公共知识库由班级管理员和超级管理员共同维护，系统中所有用户（包括所有助教）都可以使用。

### Q：我修改了公共知识库，其他用户什么时候能看到？

A：修改后即时生效。其他用户在使用知识匹配或AI助手查询时，会自动获取最新的公共知识数据。

### Q：忘记密码怎么办？

A：目前系统为本地存储方式，密码无法自行找回。您可以：
- 联系超级管理员在「用户管理」中为您重置密码
- 如果之前备份过数据，可以恢复数据后使用已保存的账号信息

### Q：如何在不同电脑上使用班级管理员功能？

A：在电脑 A 上进入「设置」→「数据与安全」→「备份数据」，将导出的 JSON 文件传输到电脑 B，然后在电脑 B 上「恢复数据」即可。
