## 1. 快速入门

### 1.1 登录系统

系统提供两种方式进入：

- **已有账号**：在登录页输入您的 **用户名** 和 **密码**，点击「登录」即可进入系统。
- **注册新账号**：点击「注册」标签，输入用户名、密码和确认密码，注册成功后自动跳转回登录页。

> 内置测试账号：
> - 助教老师：`tabuddy` / `123456`
> - 管理员：`admin` / `123456`

### 1.2 系统概览

登录后您将进入仪表盘，左侧是功能导航侧边栏，右侧是内容区域。系统主要功能包括：

| 功能模块 | 说明 |
|---------|------|
| 仪表盘 | 工作概览、快捷入口、数据看板 |
| 我的工作流 | 自定义每节课的标准化任务流程 |
| 班级管理 | 查看班级信息、上课时间、课程任务 |
| 学生管理 | 查看和管理学生信息 |
| 作业管理 | 记录和评估学生作业完成情况 |
| 随堂测验 | 记录小测成绩，跟踪正确率 |
| 反馈管理 | 生成和导出课后反馈 |
| 知识库管理 | 管理个人知识库 |
| 教学资源 | 管理课件、链接等教学资源 |
| 移动端上传 | 手机拍照上传课堂照片 |
| 帮助与反馈 | 查看帮助指南、提交反馈建议 |
| 推送日志 | 查看通知推送历史 |
| 设置 | 个人信息、系统偏好、数据安全 |

### 1.3 引导系统

首次进入系统时，会弹出一个引导窗口，展示本使用说明书供您阅读。阅读完毕后点击「我已阅读并清楚」即可关闭。之后您随时可以在「帮助与反馈」页面中重新查看。

## 2. 介绍页

每次打开浏览器或刷新页面，您会先看到系统的**介绍页**。

## 3. 仪表盘（首页）

仪表盘是您每天工作的起点，汇集了所有核心数据。它由多个功能卡片组成，为您提供一站式工作概览。

### 3.1 欢迎横幅（WelcomeBanner）

- 根据当前时间自动切换问候语，如「早上好」「下午好」「晚上好」
- 显示当前登录用户的姓名和角色
- 展示三个核心信息卡片：
  - **今日课程**：显示今日总课程数、当前正在上课的班级和剩余时间
  - **待收尾班级**：点击可快速滚动到班级待办中心
  - **重点提醒**：自动识别需重测的学生（单词正确率低于 80%），点击可查看详细名单，支持一键复制重测名单

### 3.2 班级待办中心（ClassTodoCenter）

- 按班级列出所有待处理事项
- 每个班级显示三个核心指标：作业待批改数、测验待记录数、课程反馈待完成数
- 显示工作流待办事项，点击可快速操作
- 支持标记缺勤学生（通过缺勤弹窗）
- 点击班级卡片可跳转到班级详情页
- 页面每 30 秒自动刷新数据

### 3.3 课程表与考勤（ScheduleAttendanceCard）

- 支持「今日」「本周」「本月」三种视图切换
- **今日视图**：显示当前正在上课的班级及剩余时间、今日课程列表、实时时钟
- **本周/本月视图**：日历形式展示所有课程安排
- 课程状态分为：未开始、进行中、已结束
- 支持右键菜单操作：删除课程安排、标记已签到/未签到
- 月视图以彩色圆点显示考勤状态（绿色=已签到，红色=未签到）

### 3.4 班级学习概览（ClassLearningOverview）

- 折线图展示各班级的正确率变化趋势
- 支持切换查看「所有班级平均」或单个班级数据
- 显示班级总数、学生总数、记录总数

### 3.5 快捷入口（FeatureShortcuts）

- 六个快捷入口，一键跳转到核心功能页面：班级管理、作业管理、随堂测验、反馈管理、教学资源

### 3.6 快速操作面板（QuickActions）

- 六个可拖拽排序的快捷操作按钮（支持拖拽调整顺序）
- 操作项包括：新建任务、批改作业、上传小测、生成反馈、添加资源、一键导出
- 有待办未完成时显示黄色提示条
- 底部有「智能推荐」区域，显示推荐进度

### 3.7 工作统计（WorkStatistics）

- 统计今日各类型任务的完成情况
- 显示已完成和待完成的任务数量
- 周进度条展示本周完成率百分比

### 3.8 重点关注学生（AtRiskStudentsPanel）

- 自动识别正确率低于 80% 的学生
- 基于最新的测验记录计算风险
- 三种风险标签：「单词需跟进」「语法需关注」「单词+语法需跟进」
- 点击学生可跳转到测验页面查看详情

### 3.9 专注计时器（FocusTimer）

- 内置番茄钟，支持 5/15/25/30/45/60 分钟预设时长
- 开始/暂停/重置功能
- 圆形进度条实时显示剩余时间
- 像素风格数字显示，时间变化时有抖动动画
- 完成时有脉冲闪烁特效
- 显示今日、本周、累计专注时长
- 展开可查看历史专注记录

## 4. 班级管理

### 4.1 班级列表

在侧边栏点击「班级管理」进入班级列表页，您可以：

- **查看所有班级**：以卡片形式展示每个班级的信息，包括班级名称、类型、学生人数等
- **创建新班级**：点击「创建班级」按钮，填写班级名称并选择班级类型
- **班级类型**：支持 GY、KET、PET、FCE、CAE、CPE、其他等常见课程类型
- **搜索班级**：通过搜索框快速查找班级

### 4.2 班级详情

点击某个班级卡片，进入班级详情页，包含以下功能模块：

**上课时间管理**
- 添加/编辑/删除上课时间段（选择星期几 + 开始时间 + 结束时间）
- 系统会根据上课时间自动判断当前是否有课，并在仪表盘中展示

**课程任务管理**
- 查看该班级的课程任务模板
- 系统会自动为每个上课日生成对应的任务实例
- 支持标记任务完成状态
- 自动统计今日任务完成进度

**课时进度管理**
- 系统自动记录每个班级的上课课时
- 支持手动调整课时进度
- 当某课时所有任务完成时，自动颁发课时成就

**核心数据概览**
- 学生人数
- 今日小测记录数
- 今日作业评估数
- 今日反馈记录数
- 工作流待办完成统计

**学生列表**
- 查看该班级的所有学生
- 快速添加或移除学生

## 5. 学生管理

### 5.1 学生列表

- **查看所有学生**：按班级筛选查看，一目了然

- 在上课日可以标记缺勤学生

- 支持查看历史缺勤记录

## 6. 作业管理

- 支持将作业数据导出为 Excel 或 Word 文档，方便存档或发送给家长

## 9. 工作流系统

工作流是 TABuddy 的核心特色功能，它允许您为每种课程类型定义标准化的教学流程。系统会根据工作流模板在每个上课日自动生成待办事项。

### 9.1 工作流模板

在侧边栏点击「我的工作流」进入页面：

**课程体系标签**
- 页面顶部显示五个课程体系标签：**GY、KET、PET、FCE、其他**
- 点击标签切换不同课程体系的工作流模板

**编辑模板**
- **添加节点**：为工作流添加新的任务节点
- **启用/禁用节点**：灵活调整哪些任务需要执行
- **调整顺序**：通过拖拽调整任务节点的执行顺序
- **自定义节点**：创建自定义名称和图标的节点，满足个性化需求
- **编辑节点**：修改节点的名称和图标
- **删除节点**：移除不需要的任务节点

**创建自定义模板**
- 点击「新建工作流」按钮
- 输入模板名称，选择关联的课程体系
- 创建后可自由配置所有节点
- 自定义模板会出现在标签栏末尾，鼠标悬停显示删除按钮

**重置模板**
- 如果不小心修改了系统模板，可以一键重置为默认配置

### 9.2 支持的节点类型

| 节点类型 | 说明 |
|---------|------|
| 批改作业 | 检查并评估作业完成情况 |
| 作业反馈 | 针对作业情况给出反馈 |
| 小测批改 | 批改随堂小测 |
| 记录小测学情 | 记录小测成绩和学情数据 |
| 撰写课程反馈 | 撰写课后总结反馈 |
| 发送学习内容 | 发送学习资料给学生 |
| 发送课后作业 | 布置课后作业 |
| 同步小测情况 | 同步小测结果给相关人员 |
| 发布重测名单 | 公布需要重测的学生名单 |
| 自定义节点 | 您自行定义的任务节点 |

### 9.3 待办事项自动生成

- 系统根据班级的课程类型匹配对应的工作流模板
- 在每个上课日自动生成待办事项
- 在仪表盘的「班级待办中心」可以查看和操作
- 完成任务后标记为「已完成」，系统会自动同步更新

## 10. 知识库管理

知识库是系统的「知识大脑」，您可以在侧边栏点击「知识库管理」进入页面。

### 10.1 知识条目管理

- **查看条目**：浏览所有知识条目，支持搜索和按类型筛选
- **添加条目**：填写标题、关键词、内容、类型、链接、优先级等信息
- **编辑条目**：修改已有知识条目的内容
- **删除条目**：移除不需要的条目
- **重置为默认**：一键恢复系统预置的默认条目

### 10.2 知识条目类型

| 类型 | 说明 |
|------|------|
| 链接（link） | 指向外部网址 |
| 模板（template） | 常用模板 |
| 文档（document） | 操作指南或说明 |
| 信息（info） | 通用教学信息 |

### 10.3 智能匹配

- AI 浮动助手的知识问答功能会先在您的本地知识库中匹配
- 本地无匹配时自动查询公共知识库
- 匹配算法基于关键词和优先级打分排序

## 13. AI 浮动助手（小T）

这是系统右下角的浮动聊天机器人，是您的智能教学助理。点击右下角的彩色聊天气泡按钮即可打开。

### 13.1 智能问答

- 输入问题，助手会自动匹配本地或公共知识库
- 支持查询如：「现在该做什么」「小测情况怎么样」「学习中心网址」等
- 支持远程 AI 对话（需要连接后端服务）

### 13.2 快捷操作

- **一键导航**：「打开班级管理」「打开设置」「打开作业管理」等
- **主题切换**：「切换深色模式」「切换浅色模式」
- 面板中预设了快捷问题按钮，一键提问即可

### 13.3 智能体操作（Agent）

助手支持通过自然语言直接操作业务数据：

- **「创建新班级」** → 自动引导创建班级
- **「标记张三为重点关注」** → 自动标记学生
- **「录入重测名单」** → 进入重测管理流程
- **「添加新任务：课堂小测」** → 自动添加工作流节点

操作过程中会引导您补充必要信息，操作完成后自动同步到对应功能模块。

### 13.4 上课上下文感知

- 当有班级正在上课时，助手会自动识别并显示当前班级信息
- 提供该班级的实时数据：任务完成情况、剩余时间等

### 13.5 对话管理

## 17. 移动端上传

在侧边栏点击「移动端上传」进入页面，该功能专为在手机等移动设备上快速上传课堂照片或文件而设计。

### 17.1 使用方式

1. **扫码进入**：在电脑端生成二维码，用手机扫码打开此页面
2. **授权相机**：首次使用需允许浏览器访问相机
3. **拍照/选择照片**：
   - 点击「拍照」调用手机后置摄像头拍照

- **HTTPS 要求**：必须使用 HTTPS 访问才能获取相机权限
- **会话有效期**：二维码会话有过期时间，过期后需要重新扫码

### 17.3 上传后的文件

## 22. 数据管理

### 22.1 数据存储

- **所有数据默认存储在本地**（浏览器的 localStorage）
- 无需网络连接即可使用全部功能
- 刷新页面或关闭浏览器后数据不会丢失

### 22.2 数据备份与恢复

进入「设置」→「数据与安全」：

- **备份数据**：将所有数据导出为 JSON 文件保存到电脑
- **恢复数据**：从之前备份的 JSON 文件恢复数据
- **数据迁移**：可在不同电脑间迁移数据（在电脑 A 备份 → 复制文件到电脑 B → 在电脑 B 恢复）

### 22.3 数据同步机制

- 系统采用内存缓存 + API 同步的双层架构
- 任何修改先更新本地缓存，再自动同步到服务器
- 同步有防抖机制，避免频繁请求

## 24. 常见问题

**Q：数据存在哪里？会丢失吗？**
A：所有数据存储在您浏览器的 localStorage 中，关闭浏览器不会丢失。建议定期在「设置」→「数据与安全」中备份数据。

**Q：如何在不同电脑上使用？**
A：在电脑 A 上「备份数据」导出文件，将文件传到电脑 B，然后在电脑 B 上「恢复数据」即可。

**Q：公共知识库和私有知识库有什么区别？**
A：私有知识库只有您自己能看到，可以自由添加和管理；公共知识库由管理员维护，所有用户共享。

**Q：忘记了密码怎么办？**
A：目前系统为本地存储，无法找回密码。建议在「设置」→「数据与安全」中定期备份数据，或联系管理员重置密码。

**Q：AI 助手无法回答问题怎么办？**
A：AI 助手需要连接后端服务才能使用远程 AI 对话功能。离线时，助手仍可使用知识库匹配和快捷操作指令功能。

**Q：如何上报系统 Bug 或提建议？**
A：在侧边栏点击「帮助与反馈」，选择「反馈建议」标签，填写您的反馈内容，可选上传截图，提交后开发团队会在 1-3 个工作日内回复。

**Q：为什么在手机上打开移动端上传页面用不了？**
A：请注意：① 不要在微信内打开，请点击右上角「在浏览器中打开」；② 必须使用 HTTPS 访问才能获取相机权限；③ 二维码会话过期后需要重新扫码。

**Q：工作流模板修改后有什么影响？**
A：修改工作流模板后，系统会在下一个上课日根据新模板自动生成待办事项。已生成但未完成的待办不会自动变更，您可以在仪表盘中手动处理。

**Q：通知提醒收不到怎么办？**
A：请检查：① 是否在「设置」→「系统设置」中开启了消息通知；② 浏览器是否允许了该网站的通知权限；③ 如果持续收不到，可以查看「推送日志」了解推送状态。
