## 1. 快速入门

### 1.1 登录系统

系统提供两种方式进入：

- **已有账号**：在登录页输入 **用户名** 和 **密码**，点击「登录」即可。
- **注册新账号**：点击「注册」标签，输入用户名、密码和确认密码，注册成功后自动跳转回登录页。

> 内置测试账号：
> - 助教老师：`tabuddy` / `123456`
> - 管理员：`admin` / `123456`

### 1.2 系统概览

登录后您将进入仪表盘，左侧是功能导航侧边栏，右侧是内容区域。系统主要功能包括：

| 功能模块 | 说明 |
|---------|------|
| 仪表盘 | 工作概览、快捷入口、数据看板 |
| 我的工作流 | 自定义每节课的标准化任务流程 |
| 班级管理 | 创建和管理班级，设置上课时间 |
| 作业管理 | 记录和评估学生作业完成情况 |
| 随堂测验 | 记录小测成绩，跟踪正确率 |
| 反馈管理 | 生成和导出课后反馈 |
| 知识库管理 | 管理本地和公共知识库 |
| 教学资源 | 管理课件、链接等教学资源 |
| 设置 | 个人信息、系统偏好、数据安全 |

## 2. 介绍页

每次打开浏览器或刷新页面，您会先看到系统的**介绍页**。

### 页面内容

- 左侧显示系统的名称、简介和特色功能说明
- 右侧展示设备样机图（CardSwap 动画）
- 背景有动态雪花粒子效果（PixelSnow），文字有打字机动画

### 进入系统

点击 **「进入系统」** 按钮：

- **已登录** → 直接进入仪表盘
- **未登录** → 跳转到登录页

> 注意：只有手动刷新或重新打开浏览器才会再次看到介绍页，正常使用过程中不会重复显示。

## 3. 仪表盘（首页）

仪表盘是您每天工作的起点，汇集了所有核心数据。

### 3.1 欢迎条幅（WelcomeBanner）

- 显示当前登录用户的姓名和角色
- 展示今日的核心统计：班级数、今日有课班级、待办任务数

### 3.2 班级待办中心（ClassTodoCenter）

- 按班级列出所有待处理事项
- 显示每个班级的：作业待批改数、测验待记录数、任务待完成数
- 支持快速标记缺勤学生
- 点击班级可跳转到班级详情页

### 3.3 课程表与考勤（ScheduleAttendanceCard）

- 支持「今日」「本周」「本月」三种视图切换
- 今日视图：显示当前正在上课的班级、剩余时间、今日课程列表
- 本周/本月视图：日历形式展示所有课程安排
- 支持标记调课/停课日期

### 3.4 班级学习概览（ClassLearningOverview）

- 折线图展示各班级的正确率变化趋势
- 支持切换查看「所有班级平均」或单个班级
- 显示班级总数、学生总数、记录总数

### 3.5 快捷入口（FeatureShortcuts）

- 一键跳转到：班级管理、作业管理、随堂测验、反馈管理、教学资源

### 3.6 工作统计（WorkStatistics）

- 统计今日各类型任务的完成情况
- 显示已完成/待完成的任务数量

### 3.7 重点关注学生（AtRiskStudentsPanel）

- 展示被标记为「重点」的学生名单
- 支持查看每个学生的作业、测验、反馈的详细情况

### 3.8 专注计时器（FocusTimer）

- 内置番茄钟，支持 5/15/25/30/45/60 分钟预设
- 开始/暂停/重置功能
- 显示今日、本周、累计专注时长
- 展开可查看历史专注记录

## 4. 班级管理

### 4.1 班级列表

在侧边栏点击「班级管理」进入班级列表页，您可以：

- **查看所有班级**：以卡片形式展示每个班级的信息
- **创建新班级**：点击「创建班级」按钮，填写班级名称和类型
- **设置班级类型**：支持 GY、KET、PET、FCE、CAE、CPE、其他 等类型
- **自定义班级类型**：在设置中可添加自定义类型

### 4.2 班级详情

点击某个班级卡片，进入班级详情页，包含以下功能：

**上课时间管理**
- 添加/编辑/删除上课时间段（星期几 + 开始时间 + 结束时间）
- 系统会根据上课时间自动判断当前是否有课

**课程任务管理**
- 添加每日课程任务模板
- 系统会自动为每个上课日生成任务实例
- 支持标记任务完成状态
- 自动统计今日任务完成进度

**课时进度管理**
- 自动记录每个班级的上课课时
- 支持手动调整课时进度
- 当某课时所有任务完成时，自动颁发课时成就

**核心数据概览**
- 学生人数
- 今日小测记录数
- 今日作业评估数
- 今日反馈记录数
- 工作流待办完成统计

## 5. 学生管理

### 5.3 缺勤管理

- 在上课日可以标记缺勤学生
- 缺勤数据会同步到仪表盘和工作流中
- 支持查看历史缺勤记录

- 支持将作业数据导出为 Excel 或 Word 文档

## 7. 随堂测验

### 7.1 测验记录

在侧边栏点击「随堂测验」进入页面，您可以：

- **记录测验成绩**：选择班级和学生，填写测验数据

- 系统会自动追踪重测学生的进步情况

### 7.3 班级整体正确率

- 系统根据所有测验数据自动计算班级整体正确率
- 在仪表盘的学习概览图表中展示变化趋势

### 7.4 导出功能

- 支持将测验数据导出为 Excel 或 Word 文档

## 9. 工作流系统

工作流是 TABuddy 的核心特色功能，它允许您为每种课程类型定义标准化的教学流程。

### 9.1 工作流模板

在侧边栏点击「我的工作流」进入页面，您可以：

**查看模板**
- 每个课程类型（GY、KET、PET 等）都有对应的工作流模板
- 系统为每种类型预置了默认模板

**编辑模板**
- **添加节点**：为工作流添加新的任务节点
- **启用/禁用节点**：灵活调整哪些任务需要执行
- **调整顺序**：拖拽或按钮调整任务执行顺序
- **自定义节点**：创建自定义名称和图标的节点
- **删除节点**：移除不需要的任务节点

**创建自定义模板**
- 基于现有类型创建全新的模板
- 自定义模板不受系统默认模板限制

### 9.2 工作流待办

- 系统根据当前班级的课程类型和工作流模板，自动生成每日待办事项
- 在仪表盘的「班级待办中心」可以查看和操作
- 支持标记待办为已完成

### 9.3 支持的节点类型

| 节点类型 | 说明 |
|---------|------|
| 课前考勤 | 记录学生出勤情况 |
| 作业检查 | 检查并评估作业完成情况 |
| 课堂小测 | 进行随堂测验 |
| 学习反馈 | 记录课后反馈 |
| 拍照提醒 | 提醒拍摄课堂照片 |
| 自定义节点 | 您自行定义的任务 |

## 10. 知识库管理

知识库是系统的「知识大脑」，分为私有和公共两种。

### 10.1 私有知识库

在侧边栏点击「知识库管理」进入页面：

**功能**
- **查看条目**：浏览所有知识条目
- **添加条目**：标题、关键词、内容、类型、链接、优先级
- **编辑条目**：修改已有知识条目
- **删除条目**：移除不需要的条目
- **重置为默认**：一键恢复系统预置的默认条目

**知识条目类型**
- 链接（link）：指向外部网址
- 模板（template）：常用模板
- 文档（document）：操作指南或说明
- 信息（info）：通用教学信息

### 10.2 公共知识库（管理员）

管理员专属功能，在侧边栏点击「管理」→「公共知识库管理」：

**功能**
- 与私有知识库相同的增删改查操作
- **启用/禁用**：可控制每条知识是否生效
- 数据同步到服务器（Redis），所有用户共享
- 支持重置为系统预置的默认公共知识

### 10.3 智能匹配

- 浮动助手的知识问答功能会先在本地知识库中匹配
- 本地无匹配时自动查询公共知识库
- 匹配算法基于关键词和优先级打分排序

## 13. AI 浮动助手（小T）

这是系统右下角的浮动聊天机器人，是您的智能教学助理。

### 13.1 打开助手

点击右下角的彩色聊天气泡按钮，即可打开助手面板。

### 13.2 核心功能

**智能问答**
- 输入问题，助手会自动匹配本地或公共知识库
- 支持查询：「现在该做什么」「小测情况怎么样」「学习中心网址」等
- 支持远程 AI 对话（需要连接后端服务）

**快捷操作**
- 助手支持一键导航：「打开班级管理」「打开设置」等
- 支持主题切换：「切换深色模式」「切换浅色模式」
- 预设快捷问题按钮，一键提问

**智能体操作（Agent）**
- 支持通过对话直接操作业务数据：
  - 「创建新班级」→ 自动创建班级
  - 「标记张三为重点关注」→ 自动标记学生
  - 「录入重测名单」→ 进入重测管理流程
  - 「添加新任务：课堂小测」→ 自动添加工作流节点
- 操作过程中会引导您补充必要信息
- 操作完成后自动同步到对应功能模块

**上课上下文感知**
- 当有班级正在上课时，助手会自动识别并显示当前班级
- 提供该班级的实时数据：任务完成情况、剩余时间等

### 13.3 对话管理

- 对话记录自动保存，关闭后重新打开仍可查看
- 支持清空对话历史
- 消息上限 100 条，超出后自动清理旧消息

## 14. 专注计时器

专注计时器位于仪表盘右下角，帮助您进行时间管理。

### 14.1 使用方式

1. **选择时长**：点击预设时间按钮（5/15/25/30/45/60 分钟）
2. **开始专注**：点击「开始专注」按钮
3. **暂停/继续**：点击「暂停」可随时暂停计时
4. **重置**：点击「重置」重新开始

### 14.2 数据统计

- 显示今日专注总时长
- 显示本周专注总时长
- 显示累计专注总时长
- 展开「专注记录」可查看每次专注的详细历史

### 14.3 视觉效果

- 圆形进度条实时显示剩余时间
- 像素风格数字显示
- 时间变化时数字有抖动动画
- 完成时有脉冲闪烁特效

## 15. 课时成就系统

### 15.1 成就获取

- 当某节课的所有任务都完成时，系统会自动颁发该课时的成就
- 成就会记录：班级、日期、课时数、完成时间

### 15.2 查看成就

- 在班级详情页可以查看该班级的课时成就
- 在仪表盘可以查看今日获得的成就

### 15.3 成就计算

- 系统自动追踪每个课时的任务完成情况
- 如果取消已完成的任务，成就也会自动撤销
- 下次任务全部完成时重新颁发

## 16. 通知与提醒

### 16.1 通知类型

系统会自动生成以下通知：

- **打卡提醒**：在上课时间到达时提醒您进行课堂打卡
- **任务提醒**：提醒您完成未完成的教学任务
- **班级提醒**：即将上课或下课的班级通知

### 16.2 通知操作

- 通知会显示在页面顶部的通知栏中
- 点击通知可跳转到对应的操作页面
- 支持标记为已读、已处理
- 支持一键关闭所有通知

### 16.3 去重机制

- 系统会自动记录已发送的提醒
- 避免同一提醒重复打扰您

## 19. 主题切换

TABuddy 支持深色和浅色两种主题模式。

### 19.1 切换方式

**方式一：设置页切换**
- 进入「设置」→「系统设置」
- 点击深色/浅色模式选项

**方式二：AI 助手切换**
- 打开浮动助手，输入「切换深色模式」或「切换浅色模式」

**方式三：快捷切换**
- 在浮动助手的快捷操作区域点击模式切换按钮

### 19.2 自动适配

- 主题切换后立即生效
- 刷新页面后保持之前的主题选择

## 20. 多账号管理

### 20.1 账号切换

系统支持多个账号共存，账号之间数据完全隔离。

**切换账号步骤：**
1. 点击右上角用户头像
2. 选择「切换账号」
3. 在弹出的账号列表中选择要切换的账号
4. 输入该账号的密码即可登录

### 20.2 数据隔离

- 切换账号时，系统会自动备份当前账号的所有数据
- 恢复目标账号的数据
- 每个账号的数据完全独立，互不干扰

### 20.3 账号管理

- 在「设置」→「数据与安全」中可管理账号
- 支持添加/删除已保存的账号
- 支持查看所有已注册的账号列表

## 21. 数据管理

### 21.1 数据存储

- **所有数据默认存储在本地**（浏览器的 localStorage）
- 无需网络连接即可使用全部功能
- 刷新页面或关闭浏览器后数据不会丢失

### 21.2 数据备份与恢复

进入「设置」→「数据与安全」：

- **备份数据**：将所有数据导出为 JSON 文件保存到电脑
- **恢复数据**：从之前备份的 JSON 文件恢复数据
- **数据迁移**：可在不同电脑间迁移数据（备份 → 复制文件 → 恢复）

### 21.3 公共知识库同步

- 管理员修改公共知识库后，数据会同步到服务器
- 所有用户在使用知识库匹配时，会自动获取最新数据

## ❓ 常见问题

**Q：数据存在哪里？会丢失吗？**
A：所有数据存储在您浏览器的 localStorage 中，关闭浏览器不会丢失。建议定期在「设置」中备份数据。

**Q：如何在不同电脑上使用？**
A：在电脑 A 上「备份数据」导出文件，将文件传到电脑 B，然后在电脑 B 上「恢复数据」即可。

**Q：公共知识库和私有知识库有什么区别？**
A：私有知识库只有您自己能看到；公共知识库由管理员维护，所有用户共享。

**Q：忘记了密码怎么办？**
A：目前系统为本地存储，无法找回密码。建议在「设置」→「数据与安全」中定期备份数据，或联系管理员重置。

**Q：AI 助手无法回答问题怎么办？**
A：AI 助手需要连接后端服务才能使用远程 AI 对话功能。离线时，助手仍可使用知识库匹配和快捷操作指令功能。
